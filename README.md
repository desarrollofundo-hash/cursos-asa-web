# React + Vite


📚 ASA Web - Plataforma de Cursos

ASA Web es una aplicación web desarrollada para la gestión y visualización de cursos, construida con tecnologías modernas del ecosistema JavaScript.
El proyecto utiliza React para la interfaz de usuario, Tailwind CSS para el diseño responsivo y APIs para la comunicación con el backend.

🚀 Tecnologías Utilizadas
⚛️ React
🎨 Tailwind CSS
📦 PNPM
🔗 Consumo de APIs REST
🧩 JavaScript / JSX
